# yKicchan's Slides

The slides are created using [Marp](https://marp.app/).

They are published on [GitHub Pages](https://ykicchan.github.io/slides/).

## Install

```sh
$ pnpm install
```

## New

```sh
$ pnpm new {dirname}
```

## Watch

```sh
# slides
$ pnpm dev

# tamplate
$ pnpm dev:tmp
```

## Build & Serve

```sh
$ pnpm build
$ pnpm start
```
